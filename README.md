📌 Project Overview

Wanderlust is a full-stack web application developed as a major project during my full stack development learning journey.
It is a travel listing platform where users can explore, create, and manage travel destinations with an interactive and user-friendly interface.

🚀 Features
🔐 User Authentication (Signup/Login/Logout)
🏝️ Create, Read, Update, Delete (CRUD) Listings
📍 Location-based travel listings
🖼️ Image upload for destinations
⭐ User reviews and ratings
🔎 Search and filter functionality
🧾 Flash messages and validations
📱 Responsive design for all devices
